## Ideas para la aplicación Diarios de Clases
* El alumno tendrá que matricularse en las asignaturas (que estén dadas de alta en la aplicación).
* Cada asignatura tendrá un chat para alumnos y profesores.
* Calendario que reúna todos los eventos académicos del alumno. Éste podrá incorporar sus propios eventos (tutorías del profesor, exámenes, entregas de practicas...).
---
#### Otras ideas
* Los estudiantes podrán crear un rincón de apuntes para compartir apuntes.
* Creación de test.
* Recordatorio o lista de tareas.

## Modelos que se utilizaran
* Usuario {estudiante o profesor}
* Asignatura {un profesor, estudiantes}
* Evento (para el calendario)
* Mensaje {perteneciente a un usuario}
